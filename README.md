# Atelier

Guild platform for creative commissions.

## Structure

```
guild-app/       — Main app (Next.js + Supabase + Resend)
apps/frontend/   — Secondary frontend
```

## Getting Started

```bash
cd guild-app
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16 (App Router)
- **Auth & DB:** Supabase
- **Email:** Resend
- **Payments:** Midtrans
- **Styling:** Tailwind CSS v4
